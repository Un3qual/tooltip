Tooltips

Tooltips is a simple tooltip tool (hehe) that skips the bullshit. Crafted with love using JQuery and CSS. It was designed with ease of use in mind.

Just Get Started Already!

It's super simple to start using Tooltips. Just add data-tooltip="tooltip content" to the element that you want to add the element to.
